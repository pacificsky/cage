## Pitch Template

Subject: **cage — open source CLI for sandboxing AI coding agents**

---

Hey [name],

I've been watching your videos on [specific topic] — really enjoyed [specific video or take].

I built an open source tool called **cage** that I think your audience would find useful. It isolates AI coding agents (Claude Code, Codex) in Docker containers so you can run them with full permissions without risking your host machine.

It's a single bash script — one command to start:

```
cd ~/src/my-project
cage start
claude --dangerously-skip-permissions
```

Your project is mounted at the same absolute path, so everything just works. No devcontainer config, no VM setup.

Here's a quick overview:
- GitHub: https://github.com/pacificsky/cage
- Blog post: https://pacificsky.blog/posts/2026/03/13/cage-run-ai-coding-agents-without-fear/

Happy to do a quick demo call or answer any questions. No pressure at all — just thought it might be a good fit for your channel.

Best,
Aakash

## Tips

- Personalize the first line — reference a specific recent video
- Tier 2 channels are the highest ROI for outreach (more responsive, highly targeted audience)
- Tier 1 channels are more likely to cover cage organically if it gets traction elsewhere (HN, Reddit)
- Best contact method: YouTube "business inquiries" email, Twitter/X DM, or Discord if they have one
- Follow up once after ~5 days if no response, then move on
